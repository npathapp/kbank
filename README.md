# KBank

`KBank` is a web application to teach kids about financial literacy and responsibility

### Installing the client

```bash
# navigate to project
cd client

# installing the project
npm install

# running the project
npm start
```

### Installing the server

```bash
# navigate to project
cd server

# installing the project
npm install

# running the server
npm run dev
```
